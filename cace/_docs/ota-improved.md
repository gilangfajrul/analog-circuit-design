# ota-improved

- PDK: ihp-sg13g2

## Pins

- vdd

  + Type: power
  + Direction: inout
  + Vmin: 1.45
  + Vmax: 1.55
- vss
  + Description: Analog ground
  + Type: ground
  + Direction: inout
- ibias_5u
  + Description: Bias current input 5uA (nom)
  + Type: signal
  + Direction: input
- vinp
  + Description: Voltage positive input
  + Type: signal
  + Direction: input
- vinn
  + Description: Voltage negative input
  + Type: signal
  + Direction: input
- vout
  + Description: Voltage output
  + Type: signal
  + Direction: output
